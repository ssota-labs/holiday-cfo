차트의 기본 단위. 타입은 코드 prefix(Assets/Liabilities/…)에서 온다.

## 할 수 있는 일

- `account add` / `list`
- `txn add`로 균형 거래
- `balance`로 weight 잔액 조회

## 수락 기준

1. `--cash` 계정이 cashflow opening에 포함된다
2. placeholder에는 posting 불가(트리거)
3. 불균형 txn은 거부 ([POLICY-001](/docs/domain/policy#POLICY-001))

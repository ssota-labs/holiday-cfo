외화 잔액을 원화로 보려면 **그날 관측한 환율**을 테이블에 넣는다. 이미 확정된
전표의 장부가액(weight)은 환율을 나중에 넣어도 소급해서 바꾸지 않는다.
기표할 때는 거래에 찍힌 총액(`@@`)을 우선한다.

## 수락 기준

1. 환율을 추가해도 과거 전표의 weight는 그대로다
2. `fx show`가 어떤 환율을 썼는지(종류·근거)를 보여 준다
3. 마감 때 화폐성 외화 계정을 재평가한다

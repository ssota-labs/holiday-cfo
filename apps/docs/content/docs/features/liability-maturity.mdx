부채 계정(카드·할부·대출·미지급)은 지금처럼 **상품별로** 둔다. 유동·비유동 구분은
폴더가 아니다. 기준일을 정한 뒤 **그때그때 계산하는 보고**다.

## 언제 보나

1. **물어볼 때** — 잔액 조회에 부채 만기 구분이 붙는다
2. **월 마감할 때** — 마감·미리보기 결과에 같은 합계가 붙는다

## 나누는 감각

| 부채 | 유동 | 비유동 |
|---|---|---|
| 카드·미지급 | 잔액 전액 | 없음 |
| 할부·대출 | 기준일 다음날부터 12개월 안에 갚을 원금 | 그 뒤 원금 |
| 예측 없음 | 잔액 전액 | 없음 |

총액은 장부 부채 잔액과 같다. 이자 줄은 구분에 넣지 않는다.

## 수락 기준

1. 유동 + 비유동 = 부채 총액
2. 계정 차트에 유동/비유동 루트를 만들지 않는다
3. 구분만으로 전표를 쓰지 않는다
4. 마감과 조회가 같은 규칙을 쓴다

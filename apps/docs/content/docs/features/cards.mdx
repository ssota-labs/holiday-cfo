카드 부채 계정에 청구주기를 붙이면 cashflow가 결제일에 출금을 투영한다.

## 수락 기준

1. 청구·결제일이 규칙대로 계산된다
2. 이전 주기 결제가 당기 청구를 상쇄하지 않는다 ([POLICY-004](/docs/domain/policy#POLICY-004))
3. 마감·미결제 청구가 런웨이에 남는다 ([POLICY-008](/docs/domain/policy#POLICY-008))

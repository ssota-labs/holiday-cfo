통장·카드 명세서와 장부 잔액을 맞춘 뒤, 그 달을 **마감**한다. 마감이 끝나면
외화 재평가·스냅샷이 남고, 이후에는 그 기간을 함부로 고치지 않는다.

## 흐름

1. **잔액 대조** (`assert`) — 명세서 숫자와 장부가 같은지
2. **마감** (`close`) — 미분류(대기) 전표가 없고 대조가 통과해야 함
3. **검증** (`verify`) — 감사 체인·불변식이 깨지지 않았는지

## 수락 기준

1. 명세서와 잔액이 다르면 마감하지 않는다
2. 기간 안에 분류 대기(draft)가 있으면 마감하지 않는다
3. 마감 시 외화 재평가 전표·스냅샷이 생기고, 기간 상태가 닫힘으로 바뀐다
4. `--dry-run`은 아무것도 쓰지 않는다
5. `verify`가 감사 체인과 균형 불변식을 검사한다

**As a** 1인 사용자  
**I want** 복식 다리로 거래를 남기고 싶다  
**So that** 잔액이 weight 합 0을 지킨다

### 수락 기준

1. weight 합 ≠ 0 이면 unbalanced (exit 1)
2. 기능통화 posting은 units=weight (POLICY-002)
3. --draft 는 잔액에서 제외

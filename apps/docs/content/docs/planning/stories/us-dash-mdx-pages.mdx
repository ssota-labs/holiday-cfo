**As a** 채팅으로 장부를 돌리는 1인 사용자  
**I want** 장부 화면을 여러 마크다운 페이지로 나누어 문장과 표·칸을 섞어 쓰고,
금액·잔액·현금 여유는 장부에서 구운 스냅샷만 보고 싶다  
**So that** 내 상황에 맞는 구성으로 읽을 수 있으면서도, 어제 적은 금액이
오늘 숫자처럼 남아 나를 속이지 않는다

### 수락 기준

1. 화면 폴더에 페이지 파일을 두 개 이상 두면, 각각을 구분해 본다
2. 페이지에는 허용된 칸(현금 여유, 잔액표, 장부 건강, 분류 대기 등)과
   숫자 없는 메모 문장만 넣는다
3. 페이지 본문에 금액을 직접 적어도 그 값이 “현재 잔액”처럼 권위 있게
   렌더되지 않는다
4. 장부를 바꾼 뒤 스냅샷만 다시 구우면, 페이지 배치는 그대로이고 숫자만
   바뀐다
5. 로컬 화면에서는 분류 대기 건을 클릭으로 처리하고, 배포 화면에서는
   쓰기가 되지 않는다

**As a** 채팅으로 장부를 돌리는 1인 사용자  
**I want** 플러그인을 올린 뒤 장부 지침·대시보드에 무엇이 필수로 바뀌었는지
에이전트가 읽어 알려 주고 선택 사항은 권장만 해 줬으면 한다  
**So that** 스킬만 새것이고 옆 문서는 옛말인 채 남지 않으며 내가 고친 문장은
함부로 지워지지 않는다

### 수락 기준

1. 복제본에 영향을 주는 변경이 버전별 쪽지에 **필수 / 선택**으로 나뉘어 있다
2. 에이전트가 아직 반영하지 않은 쪽지를 읽어 사용자에게 요약한다
3. 필수 항목은 빠뜨리지 않고 알리고 반영·보류 중 결정을 받는다
4. 선택 항목은 짧게 제안하며 같은 세션에서 반복 재촉하지 않는다
5. 대상 파일을 내가 손댄 경우, 통째로 덮기 전에 안내·확인이 있다
6. 세션이 열렸다는 것만으로 장부 `AGENTS.md`·dash 사용자 파일이 자동 덮이지 않는다
7. 원장 DB와 dash 메모·조종석 레이아웃은 기본 맞춤 대상이 아니다

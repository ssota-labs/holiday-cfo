**As a** 1인 사용자  
**I want** 신고·납부한 세금의 소득금액·과세표준·세액을 장부에 남기고 싶다  
**So that** 연도별 세무 이력을 PDF 없이 조회할 수 있다

### 무엇을 남기는가

신고 한 건마다 두 가지를 함께 저장한다. 자세한 칸 목록은
[prd-tax-return-sor](/docs/planning/prds/prd-tax-return-sor)를 본다.

1. **신고 표지** — 신고서 종류(종합소득세), 귀속연도, 신고일, 몇 번째 신고(최초/수정), 메모
2. **세액표 숫자** — 소득금액, 과세표준, 산출세액, 세액공제, 기납부세액, 납부할세액 등
   신고서에 **이미 적혀 있는 칸**의 값

이 숫자로 세금을 다시 계산하지 않는다. 신고서/홈택스에 보이는 값을 그대로
받아 둔다.

### 수락 기준

1. 표지와 세액표 숫자를 **한 번에** 저장한다. 표지만 남거나 숫자만 남는 중간 상태는 없다
2. 저장하는 금액·세율은 **신고서에 찍힌 값만**이다. 세액을 계산하거나 “틀린 것 같다”며 고치지 않는다
3. 귀속연도로 조회하면 소득금액·과세표준·납부할세액 등 주요 칸을 읽을 수 있다
4. 같은 신고서 종류·같은 귀속연도·같은 차수를 다시 넣으면 **거절**한다 (실수로 두 번 넣지 않도록)
5. 수정신고는 이전 기록을 지우지 않는다. **새 차수를 추가**하고, 이전 차수는 “대체됨”으로 표시한다
6. 신고서 저장만으로 통장/비용 **전표가 생기지 않는다**. 실제 납부는 기존처럼 **세금 비용·납부 전표**로 따로 한다

**As a** 1인 사용자  
**I want** 신고·납부한 종합소득세·부가가치세의 세액표 숫자를 장부에 남기고 싶다  
**So that** 연도·과세기간별 세무 이력을 PDF 없이 조회할 수 있다

### 무엇을 남기는가

신고 한 건마다 두 가지를 함께 저장한다. 자세한 칸 목록은
[prd-tax-return-sor](/docs/planning/prds/prd-tax-return-sor)를 본다.

1. **신고 표지** — 신고서 종류(종합소득세 / 부가가치세), 귀속연도,
   과세기간(부가세: 1·2기 예정·확정), 신고일, 몇 번째 신고(최초/수정), 메모
2. **세액표 숫자**
   - 종합소득세: 소득금액, 과세표준, 산출세액, 세액공제, 기납부세액, 납부할세액 등
   - 부가가치세: 과세 공급가액, 매출세액, 매입세액, 납부할세액(또는 환급세액) 등  
   신고서에 **이미 적혀 있는 칸**의 값

이 숫자로 세금을 다시 계산하지 않는다. 신고서/홈택스에 보이는 값을 그대로
받아 둔다.

### 수락 기준

1. 표지와 세액표 숫자를 **한 번에** 저장한다. 표지만 남거나 숫자만 남는 중간 상태는 없다
2. 저장하는 금액·세율은 **신고서에 찍힌 값만**이다. 세액을 계산하거나 “틀린 것 같다”며 고치지 않는다
3. 종합소득세는 귀속연도로, 부가가치세는 귀속연도+과세기간으로 조회하면 주요 칸을 읽을 수 있다
4. 같은 신고서 종류·같은 귀속연도·같은 과세기간(부가세)·같은 차수를 다시 넣으면 **거절**한다
5. 수정신고는 이전 기록을 지우지 않는다. **새 차수를 추가**하고, 이전 차수는 “대체됨”으로 표시한다
6. 신고서 저장만으로 통장/비용 **전표가 생기지 않는다**. 실제 납부는 기존처럼 **세금 비용·납부 전표**로 따로 한다

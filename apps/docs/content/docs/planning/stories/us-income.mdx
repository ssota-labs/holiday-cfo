**As a** 1인 사용자  
**I want** 업체별 수입을 등록하고 원천징수·4대보험·부가세를 맞추고 싶다  
**So that** 미도래 실수령만 투영되고, 실제 지급은 법정 공제와 대사된다

### 수락 기준

1. 오늘 이전 발생은 투영하지 않는다 (POLICY-017)
2. 입금 계정이 `--cash`가 아니면 gap으로 알리고 런웨이에서 제외한다
3. `business_withholding`은 3.3% (POLICY-018)
4. `salary`는 `--earned-tax` 없이 거부 (POLICY-019)
5. `income check`가 손편집 공제를 잡는다 (POLICY-020)

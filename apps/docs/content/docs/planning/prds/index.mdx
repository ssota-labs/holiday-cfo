개별 PRD는 사이드바에 펼치지 않는다 — 아래에서 고른다.

활성 PRD가 없으면 Vision과 Features가 현재 제품의 기준이다.
ID는 긴 설명형(`prd-tax-return-sor`)이고, 배지 티커는 짧은 키(`prd-001`)다.
다음 주기에 긴 ID PRD를 추가하고 `stories[]`로 US를 연결한다. 티커는 `prd-NNN`으로 붙인다.

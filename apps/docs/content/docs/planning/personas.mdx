## Human (1인 CFO)

- 자신의 통장·카드·대출을 기록한다
- 명세서·캡쳐를 보고 금액을 확정한다
- `review accept/reject`, `assert`, `close`의 최종 판단자
- 에이전트에게 모호하면 **물어보게** 한다 (추정 기표 금지)

## Agent (holiday-cfo skill)

- 채팅에서 원장 작업을 대행한다
- `account list` / `balance`로 오리엔트한 뒤 `--json`으로 명령한다
- OCR을 하지 않는다 — 파싱된 JSON을 `ingest submit`에 넘긴다
- 금액·계정·날짜를 지어내지 않는다
- 스킬 계약: [agent/skill](/docs/agent/skill)

## 권한 경계

| 행위 | Human | Agent |
|---|---|---|
| 원장 초기화 | 승인 후 | 실행 가능 (확인 필수) |
| draft 승인 | 필수 | 제안만 |
| assert / close | 승인 | dry-run·준비 |
| 임의 환율·수수료 추정 | 금지 | 금지 (관측값만) |

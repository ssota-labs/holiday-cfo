## `--data-file` 스키마

금액은 **소수 문자열**. `number` 금지. 키 레지스트리·필수 칸은
[plan-tax-return-sor](/docs/development/plans/plan-tax-return-sor)와
[tax 인덱스](/docs/spec/cli/tax#tax-return-add)가 기준이다.

- `kr_global_income` / `global_income`: `income_amount`·`tax_base`·`payable_within_deadline` 필수
- `kr_vat` / `vat`: `taxable_supply`·`output_tax`·`input_tax` 필수,
  `payable_tax` \| `refundable_tax` 중 최소 하나
- CLI는 라인 간 산술 관계를 검증·수정하지 않는다 (관측 SoR)

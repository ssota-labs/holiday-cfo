관련 기능: [income](/docs/features/income)

| 명령 | 설명 |
|---|---|
| `income add` | 정기수입을 등록한다. 입금 계정이 `--cash`여야 현금흐름에 잡힌다. |
| `income list` | 활성 정기수입을 나열한다. |
| `income source add` | 수입 원천과 법정 공제 regime을 등록한다. |
| `income settle` | 지급일의 세전 총액으로 법정 공제 라인을 만들고 저장한다. |
| `income check` | 정산 라인·실수령을 법정 재계산과 비교한다. 어긋나면 exit 1. |

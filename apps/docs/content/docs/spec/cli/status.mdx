대상은 `.holiday/ledger.db`가 있는 프로젝트 루트의 `status.md`다. 기존 파일은
손편집을 보존하지 않고 통째로 덮어쓴다. 본문에는 자산 잎 계정과 카드·대출·할부·
그 외 부채의 이름 및 계정 코드만 들어가며, 잔액과 현금흐름 숫자는 넣지 않는다.

파일을 고쳐도 원장은 바뀌지 않는다. 같은 원장 상태에서 다시 실행하면 갱신 시각을
제외한 목록 절은 같다.

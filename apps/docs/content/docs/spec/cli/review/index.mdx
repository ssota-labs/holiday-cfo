| 명령 | mutates | 요약 |
|---|---|---|
| [`review list`](#review-list) | no | pending draft 목록. |
| [`review accept`](#review-accept) | yes | draft를 posted로 승격한다. 이미 균형이므로 균형 실패는 없다. |
| [`review reject`](#review-reject) | yes | draft를 거부한다. ingest 행은 중복 기억용으로 유지한다. |

SoR는 로컬 `.holiday/ledger.db`(SQLite). Postgres 엔진은 동일 논리 스키마를 2개 마이그레이션으로 squash하며, CLI는 열지 않는다.

`book.schema_version` 상수(2)와 **마이그레이션 폴더 수(5)** 를 혼동하지 말 것.

## 사실 / 예측 / 파생

| 종류 | 예 | 저장 |
|---|---|---|
| 사실 | txn, posting, fx_rate, assertion, tax_return(+line) | 저널·테이블 |
| 예측 | card, installment, recurring, recurring_income, loan schedules | 저널 밖 |
| 파생 | balance, cashflow | 조회 시 집계 |

## 테이블 요약

| 테이블 | 역할 |
|---|---|
| book | 싱글톤 메타 (functional_currency, close_grain, timezone) |
| commodity / account | 레지스트리·차트 |
| txn / posting | 저널. posting PK (txn_id, seq) |
| period | 마감 구간 |
| card, installment(+row), recurring, recurring_income, loan(+row) | 스케줄 |
| fx_rate | 환율 |
| ingest_batch / ingest_item | draft 게이트 |
| balance_assertion | 외부 대사 |
| snapshot / snapshot_balance | 마감 스냅샷 |
| command_log | idempotency |
| audit_log | 해시 체인 |
| __holiday_migrations | 이름+sha256 ([POLICY-016](/docs/domain/policy#POLICY-016)) |

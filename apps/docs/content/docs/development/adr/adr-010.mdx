>
신고서에 찍힌 숫자는 `tax_return` / `tax_return_line`에 둔다. 거래 저널과
**떨어진 사실 테이블**이다. 앞으로의 스케줄도 아니고, 마감 스냅샷도 아니다.
1차 form은 종합소득세(`kr_global_income`)와 부가가치세(`kr_vat`)다. 부가세는
연도만으로 한 건을 특정할 수 없어 `period`(1·2기 예정·확정)를 표지에 둔다.

세금을 **낸** 현금 이동은 예전처럼 `Expenses:Tax:*` 전표다. 신고서 기록은
“얼마를 신고·결정했다고 적었는가”에만 답한다. 세율·공제 한도로 세액을
다시 계산하지 않는다.
</Decision>

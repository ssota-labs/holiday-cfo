>
분류 대기 큐처럼 **클릭으로 쓰는** 화면이 생기면서 경계를 나눴다.

- **로컬 dash (`pnpm dev`)** — 조종석. `/api/holiday/*`가 CLI를 셸로 부른다.
  클릭 한 번이 `holiday review accept --category …`다. 검증과 감사 체인은
  여전히 CLI 것이다.
- **배포 dash (정적 호스트)** — 사진. 서버가 없어 쓰기 API도 없다. 블록이
  쓰기를 시도하다 실패하면 안내 한 줄로 접힌다. **실패 자체가 보안 경계다.**

에이전트가 만지는 카탈로그에는 액션도 금액 prop도 없다. 쓰기가 일어나려면
사용자가 **자기 기계**에서 버튼을 눌러야 한다.
</Decision>

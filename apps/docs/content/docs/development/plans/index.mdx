특정 변경을 완료하기 위한 기간성 문서다. 개별 계획은 사이드바에 펼치지 않는다 — 아래에서
고른다.

## 상태

- `draft`: 검토 전
- `ready`: 구현을 시작할 수 있음
- `active`: 구현 중
- `done`: 구현과 검증 완료
- `superseded`: 다른 계획으로 대체

구현 PR은 base 브랜치에 `ready` 또는 `active` 상태로 존재하는 계획만 참조할 수 있다.
